# BuildStuff2023

Slides and sample code for OpenAPI talk at Build Stuff 2023

## DunDat example API

Documentation for the DunDat API is built in GitHub Actions from the OpenAPI YAML spec.

It is available in both
[Elements](https://stoplight.io/open-source/elements) and
[Redoc](https://github.com/Redocly/redoc) formats:

- [Elements docs](https://rendlelabs.github.io/BuildStuff2023/dundat/elements.html)
- [Redoc docs](https://rendlelabs.github.io/BuildStuff2023/dundat/redoc.html)
